# PureMediator.Net

**PeneMediator.Net** is a lightweight, extensible implementation of the Mediator pattern for .NET applications.  
It enables clean separation of concerns by routing requests through a single mediator and supports features like:

- ✔ Request/Response messaging  
- ✔ Pipeline Behaviors (logging, validation, caching, transactions, etc.)  
- ✔ Built-in validation support  
- ✔ Easy dependency injection with Microsoft.Extensions.DependencyInjection  
- ✔ Fully extensible and open for customization  
- ✔ Simple API, minimal abstractions
